# Contributing to 6MM Public Repositories

Thank you for helping improve 6MM public resources.

6MM public repositories are maintained as official documentation, integration,
and community-facing surfaces. Contributions are reviewed before they are
merged.

## Before Opening a Pull Request

- Keep changes accurate, implementation-focused, and aligned with the public product surface.
- Do not include credentials, tokens, private keys, customer data, partner-specific terms, or internal-only operational details.
- Update related documentation, links, and translations when applicable.
- Run the relevant validation command for the repository.

## Pull Request Review

Maintainers may request review from product, engineering, security, or legal
stakeholders depending on the content area.

By submitting a contribution, you agree that 6MM may use, edit, publish, and
distribute that contribution as part of official 6MM materials.
