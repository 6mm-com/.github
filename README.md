# 6MM GitHub Profile

This repository powers the public GitHub organization profile for 6MM.

The rendered organization profile is maintained in:

```text
profile/README.md
```

GitHub displays that file on the public overview page for:

```text
https://github.com/6mm-com
```

## Maintenance

Keep the profile concise, official, and aligned with the public website and
documentation:

- Website: https://www.6mm.com
- Documentation: https://docs.6mm.com
- App: https://app.6mm.com

This repository also provides organization-level community health defaults for
future public repositories:

- `CONTRIBUTING.md`
- `SECURITY.md`
- `SUPPORT.md`
- `.github/pull_request_template.md`
